# Журнал змін

Усі важливі зміни в PyMon задокументовані в цьому файлі.

Формат базується на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
проєкт дотримується [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [2.1.0] - 2026-05-10

### 🏗️ Великий рефакторинг — Модульна архітектура та Async Engine

#### Додано
- **Модульна API архітектура**: Розділено монолітний `endpoints.py` на структуровані роутери (`auth`, `servers`, `metrics`, `alerts`) в `pymon/api/routers/`.
- **Розділення фронтенду та бекенду**: Весь HTML/JS/CSS винесено в `pymon/templates/` та `pymon/static/`.
- **Асинхронний Scrape Manager**: Повністю перероблено рушій збору метрик з використанням `asyncio` та `httpx.AsyncClient`.
- **Уніфікований шар бази даних**: Централізована ініціалізація БД в `pymon/database.py`.

#### Змінено
- **CLI**: Модернізовано `cli.py` для підтримки нової модульної структури.
- **Розгортання агентів**: Оновлено команди `node_exporter` та `windows_exporter`.
- **Продуктивність**: Оптимізовано парсинг метрик та логіку збереження для Linux та Windows.

#### Видалено
- Legacy "God Object" файли: `web_dashboard.py`, `web_dashboard_enhanced.py`, `web_dashboard_simple.py`, `web_ui.py`.

## [2.0.0] - 2026-03-19

### ✨ Новий дашборд у стилі Grafana

#### Додано
- **Grafana-подібна темна тема** з налаштуванням світлої/темної теми.
- **Chart.js** для графіків реального часу.
- **Time Range Selector** (5m, 15m, 1h, 6h, 12h, 24h, 3d, 7d).
- **Експорт даних** у CSV та JSON.
- **Disk Breakdown**: Детальна інформація про кожен тому.
- **Uptime Timeline**: Відсоток аптайму за період.
- **Compare Endpoints**: Порівняння метрик між серверами.

#### Виправлено
- Балансування навантаження між API та WebSocket.
- Кешування даних.
